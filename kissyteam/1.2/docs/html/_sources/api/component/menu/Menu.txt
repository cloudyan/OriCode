.. attribute:: Menu.prototype.highlightedItem

    {Menu.Item} - 当前高亮的儿子菜单项.
    
.. hint::

    隐藏当前菜单的子菜单只需要将该属性置 null 即可
        
    .. code-block:: javascript
    
        menu.set("highlightedItem",null);
