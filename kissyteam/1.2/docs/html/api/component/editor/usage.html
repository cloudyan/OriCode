初始化¶

编辑器弹窗 z-index 设置¶

KISSY.Editor.config.baseZIndex¶

类型 number, 默认 10000. 编辑器内弹窗z-index下限.

Note

为防止和其他页面元素互相覆盖, 请保证这个数字整个页面最高

例如:

// 编辑器内弹窗z-index下限, 防止互相覆盖, 请保证这个数字整个页面最高
KISSY.Editor.Config.baseZIndex = 10000;

构造器¶

class KISSY.Editor¶
KISSY.Editor (textareaId,config)
产生编辑器实例对象
Parameters:
 
  • textareaId (string) – 编辑器装饰的文本输入框 id, 注意 id前加 #
  • config (Object) –

    编辑器以及其插件的配置项. 包括：

    config.attachForm¶

    类型 boolean, 是否监控编辑器所属的表单提交

    config.customStyle¶

    类型 string, 编辑区域的自定义样式

    config.focus¶

    类型 boolean, 是否编辑器初始自动聚焦(光标位于编辑器内)

    config.pluginConfig¶

    类型 object, 编辑器插件的相应配置
Returns:
 

编辑器实例对象
Return type:
 

KISSY.Editor

例如：

KISSY.ready(function(S){
    S.Editor(“#textareaId”,{

        // 是否监控编辑器所属的表单提交
        "attachForm":true,

        // 自定义样式, 将自己的样式加入编辑器
        customStyle:"p{color:red;}",

        // 是否编辑器初始自动聚焦(光标位于编辑器内)
        focus:true
    }).use(plugins);
});
